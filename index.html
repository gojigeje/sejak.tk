*.sejak.tk untuk Dynamic DNS

keren.sejak.tk - Web server yang keren sejak jaman masih sekolah TK. Secara teknis, situs sejak.tk ini dihosting di server ini. Server ini selain sebagai tempat melampiaskan kesuntukan admin-nya, juga menyediakan mirror beberapa aplikasi Open Source, PDF Buku / Majalah / Tabloid Linux. Yang lebih keren lagi, server ini jalan di laptop :D

ftp.sejak.tk - linux.sejak.tk - tomboy.sejak.tk - Beberapa situs dan WebApp lain yang dihosting di server keren.sejak.tk

git.sejak.tk - Percobaan subdomain untuk halaman github menggunakan custom domain.

tampan.sejak.tk - Servernya orang yang ngaku tampan. Jalan di laptop juga.

mini.sejak.tk - sebuah server mini yang terdiri dari mini router TP-LINK TL-MR3020 yang dipasangi sistem operasi tertanam bernama OpenWrt, sistem ini berbasis Linux.

*.sejak.tk untuk URL-Forwarding

ganteng.sejak.tk - cantik.sejak.tk - Link menuju twitter orang paling ganteng dan cantik sejak jaman TK..
